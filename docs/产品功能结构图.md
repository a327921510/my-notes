# 产品功能结构图

```
My Notes（V1 MVP）
├── 1. 账号与会话
│   ├── 登录（邮箱 + 密码 → JWT）
│   ├── 退出
│   └── 未登录可本地使用（推送/拉取需登录）
│
├── 2. Web 端（apps/web）
│   ├── 2.1 笔记区（NotesPage）
│   │   ├── 文件夹管理（新建 / 重命名 / 删除）
│   │   ├── 笔记 CRUD（新建 / 编辑 / 删除 / 移动）
│   │   ├── 富文本编辑（Quill：文字 + 图片粘贴/拖拽/选择）
│   │   ├── 左侧文件夹树 + 右侧编辑区（Splitter 布局）
│   │   ├── 面包屑导航
│   │   └── 同步状态展示（SyncBadge）
│   │
│   ├── 2.2 站点信息区（SitesPage）
│   │   ├── 站点列表（左侧面板：搜索 / 新增 / 选择）
│   │   ├── 站点详情（右侧面板：条目 CRUD）
│   │   ├── 条目类型：账号 / 密码 / ID / 待办 / 自定义
│   │   ├── 手动推送到云端
│   │   ├── 从云端拉取合并
│   │   └── 冲突处理（保留两份）
│   │
│   ├── 2.3 云盘工作台（CloudDrivePage）
│   │   ├── 目录管理（新增 / 删除）
│   │   ├── 文件管理（上传 / 下载 / 重命名 / 删除）
│   │   ├── 手动推送到云端（目录 → 文件 → 二进制）
│   │   ├── 从云端拉取合并（LWW + 缺失回退 local_only）
│   │   ├── 删除同步（tombstone 机制）
│   │   └── 悬浮同步面板（SyncPanel）
│   │
│   ├── 2.4 上传中心 / 同步
│   │   ├── 推送：笔记（含图片 multipart）/ 短文本 / 站点
│   │   ├── 拉取：云端列表 → 本地合并
│   │   └── 离线导出 / 导入（JSON 归档）
│   │
│   └── 2.5 全局
│       ├── 登录入口 + 登录弹窗（MainLayout Header）
│       ├── Ant Design 中文主题
│       ├── 路由导航（笔记区 / 站点信息区 / 云盘）
│       └── 本地存储（IndexedDB / Dexie）
│
├── 3. 浏览器扩展（apps/extension）
│   ├── 短文本 CRUD（按当前页面 domain 隔离）
│   ├── 类型选择（账号 / 密码 / ID / 待办 / 自定义）
│   ├── 登录（chrome.storage.session 存 JWT）
│   ├── 手动推送 / 手动拉取
│   ├── 旧版数据迁移（chrome.storage → IndexedDB）
│   └── 短文本 → 站点结构迁移
│
├── 4. 后端 API（apps/api）
│   ├── 认证：POST /api/auth/register、POST /api/auth/login
│   ├── 笔记：POST /api/notes/push、GET /api/notes、DELETE /api/notes/:id
│   ├── 图片/文件：POST /api/upload/image、GET /api/files/:storageId
│   ├── 短文本：POST /api/snippets/upsert、GET /api/snippets、DELETE
│   ├── 站点：POST /api/sites/upsert、POST /api/sites/push-full、GET /api/sites、DELETE
│   ├── 站点条目：POST /api/site-items/upsert、GET /api/site-items、DELETE
│   └── 云盘：POST /api/drive/folders/upsert、GET /api/drive/folders
│         POST /api/drive/files/push、GET /api/drive/files
│         GET /api/drive/files/:cloudId/download、DELETE /api/drive/files/:clientFileId
│
└── 5. 共享包（packages/）
    ├── @my-notes/shared — 类型、常量、纯函数冲突判定
    │   ├── 记录类型：SyncStatus、NoteRecord、SnippetRecord、ImageRecord 等
    │   ├── 标签常量：SYNC_STATUS_LABELS、SNIPPET_TYPE_LABELS
    │   ├── 同步工具：needsUpload、nextSyncAfterEdit、createId
    │   ├── 笔记拉取判定：hasNoteConflict、decideNotePull
    │   ├── 短文本拉取判定：snippetPullShouldApply、buildSnippetRowFromCloudPull
    │   ├── 站点冲突判定：hasSiteConflict、decideSitePull、hasSiteItemConflict、decideSiteItemPull
    │   └── 云盘冲突判定：hasDriveFolderConflict、decideDriveFolderPull、hasDriveFileConflict、decideDriveFilePull
    │
    ├── @my-notes/local-db — Dexie IndexedDB 定义（Web + 扩展共用）
    │   ├── NotesDB（库名 my_notes_v1，7 个 schema 版本）
    │   ├── 表：folders / notes / images / snippets / site_spaces / clips / blobs
    │   │       sites / site_items / drive_folders / drive_files / drive_file_tombstones
    │   └── seed：ensureDefaultFolder、ensureSiteSpace
    │
    └── @my-notes/sync-client — 副作用型同步（fetch + Dexie 写入）
        ├── pullFromCloud（笔记 + 短文本 + 图片下载）
        ├── uploadNote / uploadSnippet
        ├── syncAllSitesWithConflict / pullSitesFromCloud / syncDirtySitesToCloud
        ├── uploadSite / uploadSiteItem
        ├── deleteSiteOnCloud / deleteSiteItemOnCloud
        └── joinApiPath（同源/跨域 URL 拼接）
```

## 三端共用关系

```
┌──────────────────────────────────────────────────────┐
│                  @my-notes/shared                     │
│  纯类型 + 纯函数（冲突判定、ID生成、同步状态转换）       │
│  三端共用：Web / Extension / API 均可引用               │
└──────────────────┬───────────────────────────────────┘
                   │
      ┌────────────┼────────────┐
      ▼            ▼            ▼
 ┌─────────┐ ┌──────────┐ ┌─────────┐
 │local-db  │ │sync-client│ │  API    │
 │(Dexie)   │ │(fetch+DB) │ │(Fastify)│
 │Web+Ext   │ │Web+Ext    │ │独立部署  │
 └─────────┘ └──────────┘ └─────────┘
      │            │
      ▼            ▼
 ┌─────────────────────┐
 │ Web / Extension UI   │
 └─────────────────────┘
```

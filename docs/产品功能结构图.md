## 说明

- 信息架构采用两个平级一级入口：`笔记区` 与 `站点信息区`。
- `站点信息区` 为特殊功能模块，默认与笔记数据互不干扰。
- 同步策略为**手动推送 + 手动拉取**，并支持**离线导出/导入**；默认本地优先。
- 浏览器扩展仅提供网页短文本数据 CRUD，不展示笔记目录与笔记内容。

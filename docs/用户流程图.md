# 用户流程图（Flow Chart）

## 0) Web 端：登录流程

```mermaid
flowchart TD
  A([开始]) --> B[进入 Web 端]
  B --> C{Header 登录态}
  C -->|已登录| D[显示邮箱 + 退出按钮]
  D --> Z([使用各功能区])
  C -->|未登录| E[显示 登录 按钮]
  E --> F[点击 登录 按钮]
  F --> G[弹出 LoginModal 弹窗]
  G --> H[输入邮箱和密码]
  H --> I[点击 登录 提交]
  I --> J{调用 POST /api/auth/login}
  J -->|成功| K[写入 useAuthStore token+user]
  K --> L[弹窗关闭 Header 刷新为已登录态]
  L --> Z
  J -->|失败| M[提示错误信息]
  M --> G
```

## 1) Web 端：笔记区本地编辑、手动推送与手动拉取

```mermaid
flowchart TD
  A([开始]) --> B[进入Web端]
  B --> C{选择功能区}
  C -->|笔记区| D[新建或编辑笔记 文字+图片]
  D --> E[写入IndexedDB]
  E --> F{同步操作?}
  F -->|无| G[保持本地状态 local_only/dirty]
  G --> Z([结束])
  F -->|推送上传| H{是否已登录}
  H -->|否| I[登录]
  I --> J[上传图片]
  H -->|是| J
  J --> K{图片上传成功}
  K -->|否| L[标记failed 提示重试]
  L --> Z
  K -->|是| M[上传笔记正文]
  M --> N{笔记上传成功}
  N -->|否| L
  N -->|是| O[回写cloudId 标记synced]
  O --> Z
  F -->|拉取| P{是否已登录}
  P -->|否| Q[登录]
  Q --> R[请求云端列表与资源]
  P -->|是| R
  R --> S{拉取成功}
  S -->|否| T[提示失败 本地数据保留]
  T --> Z
  S -->|是| U[合并本地与云端 更新IndexedDB]
  U --> Z
```

## 1A) Web 端：离线全量导出与导入

```mermaid
flowchart TD
  A([开始]) --> B{操作类型}
  B -->|导出| C[导出全部笔记与图片等]
  C --> D[生成归档文件]
  D --> E[用户保存到磁盘或网盘]
  E --> Z([结束])
  B -->|导入| F[选择归档文件]
  F --> G{校验版本与完整性}
  G -->|失败| H[提示错误]
  H --> Z
  G -->|成功| I{合并策略}
  I -->|合并| J[写入IndexedDB 合并]
  I -->|覆盖需确认| K[二次确认后清空再写入]
  K --> J
  J --> Z
```

## 2) Web 端：站点信息区（特殊功能）流程

```mermaid
flowchart TD
```

## 3) 浏览器扩展：短文本 CRUD + 登录与手动同步

```mermaid
flowchart TD
  A([开始]) --> B[打开浏览器扩展]
  B --> C{已登录?}
  C -->|否| D[可选：登录同一账号 as Web]
  C -->|是| E[读取当前页面URL/Domain]
  D --> E
  E --> F{用户操作}
  F -->|新建| G[录入短文本并选择类型]
  F -->|查询| H[按Domain查看条目]
  F -->|编辑| I[修改条目内容或类型]
  F -->|删除| J[删除条目]
  F -->|手动拉取| K{已登录?}
  K -->|是| L[请求云端合并入IndexedDB]
  K -->|否| M[提示先登录]
  F -->|手动推送| N{已登录?}
  N -->|是| O[上传待同步短文本等]
  N -->|否| M
  G --> P[保存到IndexedDB snippets]
  H --> Q[展示结果]
  I --> P
  J --> P
  L --> Q
  O --> Q
  M --> Q
  P --> Q
  Q --> R([结束])
```

## 4) 端间协同流程（扩展与Web）

```mermaid
flowchart LR
  A[浏览器扩展短文本CRUD] --> B[扩展侧 IndexedDB my_notes_v2 与 Web 同 schema]
  W[Web站点信息区] --> X[Web侧 IndexedDB my_notes_v2]
  B -->|同一账号 手动推送| C[后端API]
  X -->|同一账号 手动推送| C
  C -->|手动拉取| B
  C -->|手动拉取| X
```

- 两端本地库物理隔离，通过**同一 userId 云端**与**相同同步语义**保持一致。
